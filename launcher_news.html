[1.12.2] CTD Lite Update (12/04/2017)

CTD Lite has received an update. You may check out the patch notes here.

Launcher Update 1.2.4 (11/17/2017)

Launcher Update 1.2.4 is now live. It simply just changes the URL where our packs are hosted. Simple enough.

Launcher Update 1.2.2 (9/27/2017)

Launcher Update 1.2.2 is now live. It fixes the "Clear Window" button in the console (added in 1.2.1) not being localized correctly. Click Update in your launcher to receive this update.

Launcher Update 1.1.4 (8/17/2017)

Launcher Update 1.1.4 is an optional update. This update adds the ability to view both crash reports and logs for a pack by right clicking on it from the menu.

Launcher Update 1.1.2 (7/11/2017)

Launcher Update 1.1.2 is an optional update. This update changes the default RAM values to 2GB minimum from 1GB and sets some custom JVM arguments. If you wish to use these settings, please delete config.json from your launcher's directory.

1.12 Unstable Pack (6/30/2017)

The 1.12 Unstable pack is ready for user consumption! We'll have a server open later today. This pack is unstable, duh. Bugs and issues are expected. Please report any bugs to the Discord so that TMG may forward them to the mod authors.
